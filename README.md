Como instalar?

- Siga as instruções para instalar o pixi em https://pixi.sh/latest/installation
No windows, provavelmente o [MSI installer](https://github.com/prefix-dev/pixi/releases/latest/download/pixi-x86_64-pc-windows-msvc.msi) é a melhor opção.
- Se a instalação foi sucedida, dentro to diretório `Proj_Pellets_Amarelamento`, digite `pixi notebook` e um browser com o jupyter notebook deve abrir automaticamente.
